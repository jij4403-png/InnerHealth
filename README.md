# InnerHealth Standalone Web Application

이 프로젝트는 아토피 피부염 환자들을 위해 자가진단, 가상 생검 분석(마이크로바이옴 시뮬레이션), 그리고 피부 장벽 맞춤 케어 솔루션을 제공하는 웹 서비스의 독립 실행형 HTML/CSS/JS 버전입니다.

## 📂 폴더 구조

- `index.html` - 메인 웹 애플리케이션의 화면 구조 및 레이아웃을 담당하는 HTML 파일입니다.
- `css/style.css` - 고급스러운 세이지 그린/포레스트 그린 브랜딩 테마가 적용된 CSS 스타일시트입니다.
- `js/app.js` - 자가진단 계산식(SCORAD), 가상 마이크로바이옴 시뮬레이션 엔진, 그래프 렌더링, 4주 스케줄링 등 모든 동적 로직이 포함된 자바스크립트 파일입니다.

## 🚀 실행 방법

### 1. 로컬에서 단순 실행 (인터넷 연결 필요 - Chart.js 라이브러리 로드용)
- `index.html` 파일을 더블클릭하여 사용하는 웹 브라우저(Chrome, Edge, Safari 등)에서 바로 열 수 있습니다.
- 별도의 웹 서버나 설치 과정 없이 로컬에서 즉시 정상 동작합니다.

### 2. 웹상에 배포하기 (무료 배포 채널 추천)
이 정적 소스코드는 다음과 같은 플랫폼을 통해 단 몇 분 만에 온라인 배포할 수 있습니다:
- **GitHub Pages**: 이 폴더 내의 파일들을 그대로 깃허브 저장소에 업로드한 후, Repository Settings > Pages 메뉴에서 배포할 수 있습니다.
- **Netlify**: [Netlify](https://www.netlify.com/)에 회원가입 후 이 폴더 전체를 드래그 앤 드롭하면 즉시 배포 URL이 생성됩니다.
- **Vercel**: Vercel CLI 또는 GitHub 연동을 통해 손쉽게 배포할 수 있습니다.
